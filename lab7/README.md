# AniLog API - Lab 7

AniLog API builds on the Lab 6 anime and manga vault app and adds a back-end API for CRUD operations.

This folder starts from the Lab 6 client and will be extended with:

1. A JWT-protected CRUD API for saved vault entries.
2. A `/token` endpoint for short-lived access tokens.
3. Swagger UI documentation.
4. Front-end integration with the back-end.

## Development

```bash
npm install
npm run dev:full
```

## Current Ports

- Front-end: `http://localhost:5174`
- API scaffold: `http://localhost:3001/health`
